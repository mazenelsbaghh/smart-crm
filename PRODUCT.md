# Product

## Register

product

## Users
Operations teams, customer support agents, and managers who need to handle high-frequency customer chats via WhatsApp, configure workflows, and oversee AI auto-replies. They value speed above all, requiring quick responses, minimal clicks, keyboard shortcuts, and a frictionless experience.

## Product Purpose
To automate and streamline customer service operations via WhatsApp by combining Baileys WhatsApp Gateway, ASP.NET Core backend, and Gemini 3.5 Flash. It provides a real-time operations dashboard, automated smart routing, and instant responses to boost conversion rates and customer satisfaction.

## Brand Personality
Sleek, youthful, and high-energy. Bold but professional, fast-paced, and highly interactive. It represents speed, intelligence, and modern efficiency.

## Anti-references
- Clunky, slow enterprise CRMs with heavy pages, paginated data grids, and endless menus.
- Complex multi-step wizard dialogues.
- Static, lifeless interfaces without responsive transitions.
- Light themes that feel sterile and cause eye strain during night shifts.
- Over-decorative glassmorphism and side-stripe borders that distract from core messages.

## Design Principles
1. **Frictionless Velocity**: Action must feel instantaneous. Every layout is optimized for speed, placing keyboard shortcuts and command menus at the core.
2. **Neon-Drenched Midnight Vibe**: A curated dark-theme aesthetic paired with vibrant tech-accent colors (cyan, magenta, lime) that feels alive, premium, and keeps the user focused.
3. **Immediate Context (Zero Clutter)**: Keep all critical information (chats, customer profiles, AI suggestion status) on a single unified canvas. Never hide basic controls under deep nesting.
4. **Actionable Shortcuts**: Provide keyboard triggers for every primary action (e.g., `Cmd+K` for command menu, `Cmd+Enter` to send, `R` to reply, `A` to assign, `F` for follow-ups).

## Accessibility & Inclusion
- Full keyboard accessibility: Ensure all features can be navigated and executed via keyboard shortcuts.
- WCAG 2.1 AA compliant color contrast on all text and active components.
- Reduced motion options respected by using CSS variables for transition speed.
